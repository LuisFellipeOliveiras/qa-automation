# Testes Automatizados – Portfólio de Fellipe Oliveira

Este repositório contém testes automatizados escritos em Cypress para validar o portfólio do QA Fellipe Oliveira.

### ✔ O que é testado?
- Carregamento da página
- Existência das seções principais
- Presença da imagem de perfil
- Validação básica de elementos e cores

### ✔ Como rodar os testes localmente:

```bash
npm install
npm run cy:open
```

### ✔ Como rodar em modo headless:
```bash
npm test
```

GitHub Actions roda automaticamente em cada push na branch `main`.
